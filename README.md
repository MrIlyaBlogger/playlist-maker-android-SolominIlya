# Playlist Maker

Учебное приложение для поиска треков. Проект разработан в рамках сквозного курса Яндекс.Практикума.

## Технологии

- Язык: Kotlin
- UI: Jetpack Compose
- Архитектура: MVVM + Data / Domain / UI
- Навигация: Navigation Compose
- Асинхронность: Kotlin Coroutines

## Системные требования для сборки

- Android Studio Flamingo (2022.2.1) или новее
- JDK 17
- minSdk: 24
- targetSdk: 34
- Gradle: 8.2

## Сборка и запуск

1. Склонировать репозиторий:  
   `git clone https://github.com/ваш-репозиторий/playlist-maker.git`
2. Открыть проект в Android Studio.
3. Дождаться синхронизации Gradle.
4. Запустить приложение на эмуляторе или реальном устройстве (Run → Run 'app').

## Реализованный функционал

- Главный экран с меню (поиск, плейлисты, избранное, настройки)
- Экран настроек: кнопки «Поделиться приложением», «Написать разработчикам», «Пользовательское соглашение»
- Экран поиска:
  - Поле ввода с кнопками поиска и очистки
  - Отображение результатов в виде списка (LazyColumn)
  - Индикатор загрузки
  - Сообщение об ошибке при сбое поиска
  - Сброс результатов при очистке поля
- Эмуляция сервера (Storage) и репозиторий с фильтрацией без учёта регистра
- Форматирование длительности в формат mm:ss
- Тёмная тема (переключается в зависимости от системной темы)
